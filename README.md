## MIRa - Music Information Retrieval Analysis 🎵

A comprehensive AI-powered music analysis system built with FastAPI and Next.js. Extracts tempo, key, instruments, genre, mood, and more from audio files.


### Features

-  Audio Analysis**: Tempo (BPM), key detection, loudness measurements
-  AI-Powered Recognition**: Instrument identification using Hugging Face models
-  Music Intelligence**: Genre classification, mood analysis, song structure
-  Modern Web API**: RESTful FastAPI backend with automatic OpenAPI documentation
-  Real-time Processing**: Instant analysis with progress indicators
-  Beautiful UI**: Modern React frontend with drag-and-drop upload

### Tech Stack

**Backend**: FastAPI, Python 3.10+, Librosa, Transformers, NumPy  
**Frontend**: Next.js 14, React, Tailwind CSS, Axios  
**AI/ML**: Hugging Face Transformers, Audio Classification Models  
**DevOps**: GitHub Actions, pip, npm

### Quick Start
- npm install
- npm run dev

### Prerequisites
- Python 3.10+
- Node.js 18+
- FFmpeg (`brew install ffmpeg` or `apt install ffmpeg`)
